# ☁️ Cloud Essentials Knowledge

## What is Cloud Computing?
Cloud computing is the delivery of computing services — servers, storage, databases, networking, software, analytics, and intelligence — over the internet ("the cloud") to offer faster innovation, flexible resources, and economies of scale.

---

## ☁️ Cloud Service Models

| Model | Full Name | Description |
|-------|-----------|-------------|
| IaaS | Infrastructure as a Service | Provides virtualized computing resources over the internet (e.g., AWS EC2) |
| PaaS | Platform as a Service | Provides a platform to develop, run, and manage applications (e.g., AWS Elastic Beanstalk) |
| SaaS | Software as a Service | Delivers software applications over the internet (e.g., Google Workspace) |

---

## 🌍 Cloud Deployment Models

- **Public Cloud** – Resources owned and operated by a third-party provider (AWS, Azure, GCP)
- **Private Cloud** – Resources used exclusively by one organization
- **Hybrid Cloud** – Combination of public and private clouds
- **Multi-Cloud** – Using services from multiple cloud providers

---

## 🔑 Key Cloud Concepts

- **Scalability** – Ability to scale up or down based on demand
- **Elasticity** – Automatic scaling in real-time
- **High Availability** – Systems designed to operate continuously without failure
- **Fault Tolerance** – Ability to continue operating despite component failures
- **Disaster Recovery** – Strategies to recover data and systems after a failure
- **Pay-as-you-go** – Only pay for what you use

---

## 🛡️ Cloud Security Basics

- **IAM (Identity & Access Management)** – Control who can access what
- **Encryption** – Protect data at rest and in transit
- **Firewalls & Security Groups** – Control inbound/outbound traffic
- **Compliance** – Meeting regulatory standards (GDPR, HIPAA, ISO 27001)
- **Shared Responsibility Model** – Cloud provider secures the infrastructure; customer secures their data and apps

## 📦 Core AWS Services (Essentials)

- **EC2** – Virtual servers in the cloud
- **S3** – Scalable object storage
- **RDS** – Managed relational databases
- **Lambda** – Serverless computing
- **VPC** – Virtual private network in AWS
- **CloudFront** – Content delivery network (CDN)
- **IAM** – Identity and access management
- **Route 53** – DNS and domain management
